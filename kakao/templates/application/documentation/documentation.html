basic URL param

Gether Data
request URL method[POST,GET] syntax:
$this->Request->getParam(param) | same as GET method
$this->Request->getPost(param) | same as POST method
basic URL use GET method
page : to use a class of module, using same name of class name and filename
act : to use method/function in class
req : if the system want to use encrytp URL
$this->Request->getParam('page');
$this->Request->getParam('act');
$this->Request->getParam('req');
example:
index.php?page=home&act=article
this will call file home.php in modules folder
and ask this function 'article' to execute

how to use application modules?

to create variable smarty
$this->View->assign($var,$value);

to view template smarty
$this->View->toString(param);

how to use widgets modules?

to set widgets using:
$whateverWidgetName = $this->setWidgets(param);

example:
in application modules, signing widgets to templates
$this->View->assign($var,$whateverWidgetName);

how to use Helper?

declare the Helper:
$this->whateverName = $this->useHelper($class);

use the Helper
$this->whateverName->methodOfHelper();

example:
you can construct the helper as variable in application modules
function __construct(){
$this->dateHelper = $this->useHelper('dateHelper');
}
function main(){
$checkDate = $this->dateHelper->dateDiff('2012-07-08 00:00:01');
}

how to use database model?

open or close database connection
$this->open($index_connection); to open DB
$this->close(); to close

select many row
$this->fetch($query,1);

select single row
$this->fetch($query);

insert, update, delete row
$this->query($query);

example:

$qData = "SELECT * FROM social_member LIMIT 10"
$this->open(0);
$data = $this->fetch($qData,1);
$this->close();

using encoder for any purpose
urlencode64(serialize($arrays) or $strings); to encode string or array
urldecode64($strings); to decode string from urlencode64 decoder

how to use API?

base_url API : whatever.com/service/?[service=$className]&[m=$methodName]&[params]
filenaming = [className]_service.php
example:
service file:
example_service.php
request API service:
whatever.com/service/?service=example&m=checkonline
php class:
class example_service {
var $API;
function __construct($API=null){
$this->API = $API; // same as [parent::__constructor] from API.php
$this->user = $this->API->getUserOnline();
}
function checkOnline(){
print_r($this->API->getUserOnline());exit;
}
function getAccessToken(){
global $CONFIG;
$at = get_access_token($this->user->id,$CONFIG['SERVICE_KEY']);
return $this->API->toJson(1,'access',$at);
}
}
